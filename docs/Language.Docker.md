Language.Docker
---------------

### Synopsis
Docker language definition

---

### Description

Defines the Docker language within PipeScript.
This allows the Dockerfile to be generated with PipeScript.

---

Invoke-PipeScript .\PipeScript.Example.ps.Dockerfile
